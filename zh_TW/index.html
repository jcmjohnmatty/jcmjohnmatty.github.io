層級編輯器

某個寒假，我又開始玩一些比較老的電腦遊戲，甚至包括最早的 NES （像 Marble Madness，第一代薩爾達傳說 （The Legend of Zelda）等遊戲）。我認為創造個人的遊戲是一件很有趣的事情，所以，我編寫了屬於我自己的遊戲引擎，“層級編輯器”（英語：“MapEditor”）。這樣，我能開始設計像薩爾達傳說同樣類型的遊戲。以下是那個寒假我所寫的編碼。

這個軟體用Java 寫的，主要是因為它可以跨平台執行，並且可以使用Java 中內建的圖形用戶界面骨架 Swing。軟體有一個主要的繪製區域。繪製區域能加入不同類型的精靈（像環境、對手、主角）。您也能改變層中背景圖片，也可以改變這些精靈讓它們延展，轉動，和移動。您最後能把您創造的場景所有相關訊息儲存成純文檔案。

Jellyfish Simulation

這是我二年級上電腦繪圖入門課時設計的期末專題。是由我與另外一位同學設計完成的。我主要負責項目管理，研究，設計海蜇身體的編碼，還有設計海蜇身體的質地。另外一個組員主要負責準備報告，以及設計海蜇的觸手還有其他項目管理。

海蜇的身體放鬆時，模擬為半橢圓形。橢圓頂點組成一個彈簧質量系統。海蜇的推力為調整過的正弦波。我也在次正弦波上做一些預處理與後處理，就是為了確保海蜇行動方向正確，也是確保海蜇保持應有的形狀。

海蜇的觸手是一個粒子系統。粒子的度量也是可配置的。觸手成放射狀附著於海蜇身體上。海蜇身體波動時，一系列的函數會更新每條觸手的位置。

海蜇的質地以OpenGL中的質地機制設計的。

此工程被選取為我們班中最好期末工程之六。

GPGMK：GPU 動力 3D 幾何模型核心

這是一個我開發的跨平台、GPU 動力、3D 幾何模型核心。在現代電腦輔助設計系統中，幾何模型核心有很重要的身份。它們提供集合模擬演算法，及可用來執行建設性立體幾何的基本幾何類別。若沒有幾何模型核心，那麼現代幾何模型核心不可到現在的狀況。

許多現代的系統，想 AutoDesk 和 SolidWorks，依賴於一定的平台，像 Windows 作業系統那樣的平台。現代幾何模型系統頁缺少 GPU 動力的演算法，對性能確實含有限制。

我希望，透過用跨平台技術跟 GPU 能力執行這些演算法，我可以變更現代幾何模型核心的性能，並減少以來高級 CPU 的需求，像現代系統的需求。我也希望能減少依賴於一定平台的需求，也能增強開源幾何模型核心的跨平台行、實用性。

地面插座蓋子

我是自己設計這個地面插座蓋子。我設計的時候尤其考慮了別人在上面走路的時候不會踏於地面插座蓋子上面。因此，上面和寧進的screw head就跟地面是平的。還有，所有的screw hole大小都是標準的，所以很容易移動到新的地方。我也考慮了地下的wiring，因為這個蓋子不准破壞已經存在的wiring。最後，所有的holes等都有一點點的tolerance，這樣寧進screw什麼的都很容易。

為了全面得確認我圖樣的準確性，我也定期地用了 3D印表機創造蓋子。目的是，確認需要的規範被符合。
